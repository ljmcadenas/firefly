# Setup
- Extract the `FireFly` folder to you computer
- Enter the `FireFly` folder in terminal
- Run `npm i` to install required dependencies

# Serve
- Enter the `FireFly` folder in terminal
- Serve by running `ng serve`
- Visit `http://localhost:4200`

# Tesing
- Enter the `FireFly` folder in terminal
- Run `ng test`
